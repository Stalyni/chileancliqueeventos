# Chilean Clique Listening Events - v15 final visual polish

Incluye:
- Planeta Three.js con rotación, atmósfera, nubes y transición a Chile.
- Zoom cinematográfico: primero ubica Chile, luego desciende.
- Mapa SVG de Chile con Santiago interactivo.
- Panel de Santiago con hover/click y cierre automático a los 20 segundos.
- Fondo astral responsivo con estrellas, constelaciones cíclicas y estrellas fugaces.
- Logo oficial Chilean Clique en SVG.
- Animaciones finales: entrada de Chile, título, logo, Santiago, panel.
- Cursor personalizado sobre zonas interactivas.
- Sonido opcional generado por WebAudio, sin archivos externos.
- Estado automático NOW PLAYING cuando el contador llega a cero.
- Favicon, meta description y Open Graph básico.

Para probar local:

```bash
npm install
npm start
```

Para Render:
- Build command: vacío o `npm install`
- Start command: `npm start`
